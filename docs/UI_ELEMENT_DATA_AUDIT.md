## Classification Rules Applied

| Rule | Audit effect |
|---|---|
| Labels, section titles, button labels, explanatory non-personal copy | static copy unless it promises data state, limit, entitlement, or freshness. |
| Colors, rules, icons, tab chrome, card borders, decorative sparklines with no numeric claim | static design decoration. |
| Any number, grade, ticker, price, delta, timestamp, count, chart point, article, source, status, severity, subscription state, or personalized sentence | mock data unless an exact live API field and DB/generation path were found. |
| Existing endpoint alone is not enough | marked partial if the endpoint exists but lacks exact field, history proof, generated source, or DB coverage. |
| Deltas and `was` values | require real prior snapshot/history. Otherwise P0 fallback is `-` or `New`. |
| Scores | require five-dimension formula, input rows, source, and timestamp. Otherwise partial or missing. |
