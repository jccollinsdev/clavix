# Clavix Pricing

**Status:** Launch plan

**Last updated:** May 7, 2026

---

## Positioning

Clavix has two tiers at launch:

| Tier | Role |
|---|---|
| Free | Lets a user trust the product with a small portfolio. |
| Pro | Unlocks automation, scale, deeper digest output, and higher-frequency controls. |

## Feature Split

| Feature | Free | Pro |
|---|---:|---:|
| Sign up / sign in | Yes | Yes |
| Onboarding | Yes | Yes |
| Manual portfolio entry | Up to 3 holdings | Unlimited |
| Brokerage sync | No | Yes |
| CSV import | No | Yes |
| Watchlist | Up to 5 tickers | Unlimited |
| Daily digest - Brief | Yes | Yes |
| Daily digest - Standard | Yes | Yes |
| Daily digest - Verbose | No | Yes |
| Universal ticker search | Yes | Yes |
| Ticker detail | Yes | Yes |
| Methodology drill-down | Yes | Yes |
| Recent news per ticker | Last 7 days | Last 30 days |
| Score history | 30 days | Full available history |
| Manual ticker refresh | No | 5 per ticker per day |
| Grade change alerts | Yes | Yes |
| Major news alerts | Yes | Yes |
| Watchlist alerts | No | Yes |
| Macro shock alerts | No | Yes |
| Email digest of alerts | No | Yes |
| Alert severity threshold | No | Yes |
| Export portfolio data | Yes | Yes |
| Delete account | Yes | Yes |

## Price

| Plan | Price |
|---|---:|
| Free | $0 |
| Pro | $20/month |

There is no annual plan at launch. Revisit annual pricing after retention data exists.

## Trial

New users receive a 14-day Pro trial with no credit card required. On day 15, the account automatically downgrades to Free unless the user subscribes.

## Cost Structure

### One-time

| Item | Cost |
|---|---:|
| Business entity | $550 |
| Legal | $500 |
| Apple Developer Program | $99 |
| Total | $1,149 |

### Monthly Fixed

| Item | Cost |
|---|---:|
| VPS | $15 |
| Supabase | $0 at launch on current free plan |
| MiniMax | Current flat plan, verify before launch |
| Domain | ~$1 |

Resend is not part of the launch cost structure.

### Variable

| Item | Cost |
|---|---:|
| Brokerage sync provider | Usage-based; model as Pro-only cost |
| AI/news processing | Scales with active tickers and digest depth |

## Pricing Rules

- Free must remain useful enough to build trust.
- Pro must be required for automation and scale.
- Verbose digest is Pro-only.
- Brokerage sync and CSV import are Pro-only.
- Do not add an annual plan until launch retention data supports it.
