# OpenCode Startup Note

Start every Clavis session by reading, in order:

1. `AGENTS.md`
2. `docs/STATE/project_state.md`
3. `docs/STATUS/roadmap.md`

If you want a fast manual bootstrap, run `scripts/session-start.sh`.

Keep `docs/STATE/project_state.md` updated whenever the active phase, blockers, or current focus changes.
